## 🚀 Tracking Links (LIVE)

Each driver gets a short URL for their business card. When a customer visits that link, we know who brought them.

| Person | Link |
|--------|------|
| **You need to edit `lib/tracking.ts`** with real names | https://eagleeyechauffeur.com/go/joban (example) |

**How it works:**
- Customer scans business card → visits `eagleeyechauffeur.com/go/name`
- Gets redirected to `/book` with tracking cookie (30 days)
- Any booking they make in next 30 days = tracked to that driver

**To add drivers — edit `lib/tracking.ts` and change:**
```
{ slug: 'joban', name: 'Joban Sandhu', phone: '+19299196499' },
{ slug: 'mike', name: 'Mike Brown' },
{ slug: 'sam', name: 'Sam Singh' },
{ slug: 'alex', name: 'Alex Kim' },
{ slug: 'raj', name: 'Raj Patel' },
```

Replace the placeholder names with your actual drivers.
